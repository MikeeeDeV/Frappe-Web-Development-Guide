# سياسة الخصوصية — Privacy Policy

يحتوي هذا المجلد على ملفات **سياسة الخصوصية** (Privacy Policy) الخاصة بالمشروع.
تم تصميم هذه الصفحة لتوفير شفافية كاملة للمستخدمين حول كيفية التعامل مع أي بيانات على الموقع، وهي تنتمي لمشروع تعليمي وتطويري مفتوح المصدر.

## 📄 محتويات المجلد
- `policy.html` : صفحة سياسة الخصوصية وتتميز بدعم اللغتين (العربية والإنجليزية)، إضافة إلى تصميم بصري تفاعلي وحديث.

## 🎯 النقاط الرئيسية في السياسة
تُغطي صفحة السياسة عدة بنود أساسية تشمل:
1. **البيانات المجمعة:** مثل (عنوان IP، نوع المتصفح، وقت الزيارة). ولا يتم جمع أي بيانات شخصية حساسة مثل الأسماء أو البريد الإلكتروني.
2. **استخدام البيانات:** تُستخدم فقط لتحليل أداء الموقع، وتحسين المحتوى، واكتشاف الأخطاء التقنية، ولا يتم بيع أي بيانات.
3. **ملفات تعريف الارتباط (Cookies):** يتم استخدامها لتحسين التجربة، ويمكن للمستخدم التحكم بها.
4. **الاستضافة والمشاركة:** الموقع مستضاف على GitHub Pages، وتخضع بيانات الاستخدام لسياسة خصوصية GitHub. لا توجد خوادم أو قواعد بيانات خاصة.
5. **حقوق المستخدم:** يمكن للمستخدمين طلب الوصول والتصحيح والحذف وفقًا لمعاهدات حماية البيانات.

## ✨ أبرز الميزات التقنية بالصفحة
- **دعم ثنائي اللغة (Bilingual):** زر للتحويل السريع بين اللغتين (AR/EN) ويتم حفظ الإعداد المفضل للمستخدم في الـ `localStorage`.
- **مظهر عصري (Custom UI):**
  - مؤشر ماوس مخصص (Custom Cursor).
  - شريط تقدم علوي يظهر عند النزول بالصفحة (Scroll Progress Bar).
  - خلفية جذابة وتأثير غبار الورق (Noise Effect) مبرمج بـ SVG داخل CSS.
  - نصوص متحركة وتأثيرات Glitch عند التمرير على العناوين.
- **سهولة في التصفح:** روابط انتقال سريعة للأقسام المختلفة، وزر عودة ناعم لأعلى الصفحة.
